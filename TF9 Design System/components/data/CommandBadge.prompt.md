Colored badge for Terraform command names. Each command maps to a unique semantic color.

```jsx
<CommandBadge command="plan" />
<CommandBadge command="apply" />
<CommandBadge command="destroy" />
<CommandBadge command="init" size="sm" />
```

**Supported commands:** plan, apply, destroy, init, auto, validate, refresh, state, output, import, taint, untaint, force-unlock, cost.
Colors are theme-adaptive (light/dark/dim) via CSS custom properties.
