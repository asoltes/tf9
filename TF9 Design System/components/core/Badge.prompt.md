Inline status label. Use for categorizing items in tables, cards, and lists.

```jsx
<Badge variant="green" dot>success</Badge>
<Badge variant="red" dot>failed</Badge>
<Badge variant="amber" dot>partial success</Badge>
<Badge variant="blue" dot>running</Badge>
<Badge variant="purple">auto</Badge>
<Badge>default</Badge>
```

**Variants:** `default` (grey), `blue`, `green`, `amber`, `red`, `purple`, `outline`.
Use `dot={true}` to prefix with a colored status dot.
